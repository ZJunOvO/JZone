# 已知问题与限制（jzone-player）

本清单只记录已在当前代码中可复现/可推断的问题与限制。

## 1. 已确认问题

- `index.html` 引用了 `/index.css`，但原仓库缺失该文件，导致运行时 404 与构建警告

## 2. 高风险安全隐患

- `GEMINI_API_KEY` 被注入到前端 bundle 的风险：前端不应持有长期秘钥
- 运行时依赖 CDN（Tailwind CDN、importmap 指向远程 esm.sh）：存在供应链与可用性风险

## 3. 体验/逻辑限制（Demo 可接受，但产品化需处理）

- 评论时间点与 `seek()` 未统一按裁剪区间做 clamp，可能出现“跳回片段起点”的困惑体验
- 上传页“播放选段”不会在 out 点自动停止（会播放到音频结束）
- 上传与封面使用 objectURL 未释放，频繁操作可能造成内存占用增加
- 所有数据仅在内存中，刷新页面即丢失（无持久化/无后端）
